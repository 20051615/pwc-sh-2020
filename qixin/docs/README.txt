Note: you might need a VPN to access the following pages (because Google is banned in China)
1. Ensure that Chrome is installed on your computer
2. Launch Chrome, go to chrome://settings/help and note down the version number
	e.g. 84.0.4147.89
3. Follow instructions here: https://chromedriver.chromium.org/downloads
	e.g. click on "please download ChromeDriver 84.0.4147.30"
		the link you click depends on what you noted down in step 2
4. Download chromedriver_win32.zip and unzip it
5. Place chromedriver.exe and this executable (the program for downloading from qixin) under the same folder
6. Run this executable
